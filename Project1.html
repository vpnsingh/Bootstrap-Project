Database Development

SQL is used across all databases and is useful and necessary in every project that needs to store data (i.e. 99% of all).PL/SQL is a niche programming language used for writing code inside the Oracle database. It fills that role supremely well, but PL/SQL skills are only useful for a small subset of tasks (database logic) and only with a small subset of organizations (large organizations who need and can afford expensive Oracle databases)
